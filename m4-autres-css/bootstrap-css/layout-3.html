CORPS DE PAGE

Ce qui est demandé

Affichage Mobile

  • Tout :
    • marge interne (padding) : 5px
    • Modèle de taille : inclus la bordure
  • Ordre d'affichage :
    1. Entête
    2. Panneau de gauche
    3. Corps de page
    4. Panneau de droite
    5. Pied de page
  • Entête :
    • Largeur : totale,
    • marges : 0.25rem tout autour,
    • hauteur : 150px,
    • couleur : #ed732e
  • Panneaux latéraux :
    • Largeur : 100% de l'écran,
    • marges : 0.25rem tout autour,
    • couleur : #587934
    • Hauteur du panneau de droite: 300px
  • Corps de page :
    • Bordure : solide, noire et 3px d'épaisseur,
  • Pied de page :
    • Largeur : totale,
    • marges : 0.25rem tout autour,
    • hauteur : 150px,
    • couleur : #ed732e

Affichage tablette (>576px max)

  • Hérite de l'affichage mobile.
  • Ordre d'affichage :
    1. Entête
    2. Panneau de gauche + Corps de page
    3. Panneau de droite
    4. Pied de page
  • Panneau latéral gauche :
    • Largeur : 34% de l'écran,
  • Corps de page :
    • Largeur : 66% de l'écran
  • Panneau latéral droite :
    • Largeur : 100% de l'écran,

Affichage Desktop (> 992px)

  • Hérite de l'affichage tablette.
    • Panneaux latéraux : 20% de largeur chacun, couvrant la hauteur restante de l'écran
    • Corps de page : le reste de l'écran
