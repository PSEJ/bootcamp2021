CORPS DE PAGE

Ce qui est demandé

Affichage desktop

  • Tout :
    • marge interne (padding) : 5px
    • Modèle de taille : inclus la bordure
  • Entête :
    • Largeur : totale,
    • marges : 3px tout autour,
    • hauteur : 150px,
    • couleur : #ed732e
  • Panneaux latéraux :
    • Largeur : 20% de l'écran,
    • marges : 3px tout autour,
    • couleur : #587934
  • Corps de page :
    • Bordure : solide, noire et 3px d'épaisseur,
  • Pied de page :
    • Largeur : totale,
    • marges : 3px tout autour,
    • hauteur : 150px,
    • couleur : #ed732e

Affichage tablette (900px max)

  • Hérite de l'affichage desktop.
  • Panneau latéral gauche :
    • Largeur : 34% de l'écran,
  • Panneau latéral droite :
    • Largeur : 100% de l'écran,
    • Hauteur : 300px,
  • Corps de page :
    • Largeur : 66% de l'écran

Affichage mobile

  • Hérite de l'affichage tablette.
  • Panneau de gauche, ensuite corps de page ensuite panneau de droite :
    • Largeur : 100% de l'écran,
