Affichage Mobile

  • Tout :
    • marge interne (padding) : 5px
    • Modèle de taille : inclus la bordure
  • Ordre d'affichage :
    1. Ent≖te
    2. Panneau de gauche
    3. Corps de page
    4. Panneau de droite
    5. Pied de page
  • Entête :
    • Largeur : totale,
    • marges : 3px tout autour,
    • hauteur : 150px,
    • couleur : #ed732e
  • Panneaux latéraux :
    • Largeur : 100% de l'écran,
    • marges : 3px tout autour,
    • couleur : #587934
  • Corps de page :
    • Bordure : solide, noire et 3px d'épaisseur,
  • Pied de page :
    • Largeur : totale,
    • marges : 3px tout autour,
    • hauteur : 150px,
    • couleur : #ed732e

Affichage tablette (>576px max)

  • Hérite de l'affichage mobile.
  • Ordre d'affichage :
    1. Ent≖te
    2. Panneau de gauche + Corps de page
    3. Panneau de droite
    4. Pied de page
  • Panneau latéral gauche :
    • Largeur : 34% de l'écran,
  • Corps de page :
    • Largeur : 66% de l'écran
  • Panneau latéral droite :
    • Largeur : 100% de l'écran,
    • Hauteur : 300px,

Affichage Desktop

  • Hérite de l'affichage tablette.
    • Panneaux latéraux : 20% de largeur chacun,
    • Corps de page : le reste de l'écran
